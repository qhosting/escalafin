### **2. Configurar Usuario**
```bash
git config user.name "Tu Nombre"

```

### **3. Agregar Archivos**

### **4. Crear Commit Inicial**
```bash
git commit -m "🚀 Initial commit: EscalaFin MVP v1.0.0
```
